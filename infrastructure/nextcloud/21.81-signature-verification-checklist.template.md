# eSign Verification Checklist

- [ ] Signature option selected and recorded per document type
- [ ] Approval chain completed with explicit `Approve` or `Reject` actions
- [ ] Final immutable PDF created prior to signing
- [ ] Hash manifest generated and stored in `99-Evidence-Pack`
- [ ] Signature verification report generated (options B/C)
- [ ] Certificate serial/issuer captured in evidence (options B/C)
- [ ] Independent signature validation completed on separate workstation
- [ ] `04-Signed` and `99-Evidence-Pack` sealed read-only
- [ ] No secrets/tokens captured in evidence artifacts
