# 81.160 VM Access Evidence Checklist

- [ ] Admin workstation inventory completed (`admin-workstation-inventory.csv`).
- [ ] Admin key inventory completed with fingerprints.
- [ ] Output captured: `sudo ufw status numbered`.
- [ ] Output captured: `sudo ufw status verbose`.
- [ ] Output captured: `sshd -T | egrep 'passwordauthentication|permitrootlogin|port'`.
- [ ] Output captured: proof of per-user key auth (`ssh -vv <user>@<vm-ip>` first lines).
- [ ] Optional screenshot: VS Code Remote-SSH successful connection.
