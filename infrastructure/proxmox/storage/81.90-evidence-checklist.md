# 81.90 Storage Verification Evidence Checklist

- [ ] Output captured: `pvesm status`
- [ ] Output captured: `df -hT`
- [ ] Output captured: `lsblk -o NAME,SIZE,TYPE,FSTYPE,MOUNTPOINT,MODEL`
- [ ] Output captured: `dmesg -T | tail -n 80` with no storage errors.
- [ ] LVM-thin proof captured (`lvs -a`, `vgs`, `pvs`) OR ZFS proof captured (`zpool status`, `zpool list`, `zfs list`).
- [ ] Free-space thresholds documented (warn and critical) with datastore values.
- [ ] VM I/O smoke note captured for representative VM.
- [ ] Snapshot create/rollback proof captured (snapshot name, VM, timestamps, result).
- [ ] Screenshot captured: Datacenter Storage status.
- [ ] Screenshot captured: VM snapshot view before and after rollback.
