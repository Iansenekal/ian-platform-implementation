# 81.20 Evidence Checklist

Use this checklist during the 81.20 install run and store outputs under your client evidence location.

## Required Screenshots
- [ ] Installer disk selection screen (target disk clearly visible)
- [ ] Installer management network screen (hostname/IP/gateway/DNS)
- [ ] First successful web UI dashboard login (`https://<ip>:8006`)

## Required Command Evidence
Run and save output from:

```bash
ip a
ip r
cat /etc/hosts
cat /etc/resolv.conf
ping -c 2 <gateway>
ping -c 2 <dns>
```

## Host Verification Script Output
Run:

```bash
infrastructure/proxmox/install-hardening/81.20-first-boot-verify.sh \
  --env-file infrastructure/proxmox/install-hardening/81.20-install-inputs.env
```

Store:
- [ ] Full script stdout/stderr log

## Install Variables Record
- [ ] Hostname (FQDN)
- [ ] Management IP/CIDR
- [ ] Gateway
- [ ] DNS server
- [ ] Install disk model/size/serial
- [ ] Admin alert email
- [ ] Root password vault reference
