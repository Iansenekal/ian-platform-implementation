# 81.40 Networking Design Gate Checklist

- [ ] vmbr0 baseline documented and values approved
- [ ] Management separation method selected (dedicated VLAN or same-subnet allowlist)
- [ ] Admin workstation allowlist approved
- [ ] VLAN IDs/subnets finalized if VLAN mode is enabled
- [ ] Switchport model confirmed (access/trunk)
- [ ] Firewall policy confirms no WAN exposure
- [ ] Internal DNS/routing assumptions documented
