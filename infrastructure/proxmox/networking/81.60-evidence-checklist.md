# 81.60 Network Verification Evidence Checklist

This checklist is a mandatory gate after 81.50 implementation.

- [ ] Saved output files from `81.60-network-verification.sh`.
- [ ] `summary.txt` includes all host checks as pass.
- [ ] Host state outputs captured:
- `hostname -f`
- `ip -br addr`
- `ip r`
- `ip -br link`
- `/etc/resolv.conf`
- `/etc/network/interfaces`
- [ ] Host ping outputs captured for gateway, DNS, and known LAN host.
- [ ] Host DNS resolution outputs captured (`dns01`, Proxmox FQDN, reverse lookup).
- [ ] Bridge/MAC/IP sanity outputs captured (`bridge link`, `ip link show`).
- [ ] Proxmox UI screenshot showing `vmbr0` config.
- [ ] VM MAC uniqueness confirmation note from Proxmox UI.
- [ ] Representative VM ping + DNS outputs captured (or documented skip reason).
- [ ] Written firewall/router note proving no WAN NAT/port-forward to management ports.
