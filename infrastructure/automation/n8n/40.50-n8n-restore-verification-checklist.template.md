# 40.50 n8n Restore Verification Checklist

- Service starts and remains healthy.
- SSO login and admin MFA enforcement are operational.
- Expected workflows are present after restore.
- Credential decryption works with restored key material.
- Representative workflow execution succeeds under least privilege.
- RBAC deny tests block non-admin admin-function access.
- Logs and audit pipeline remain active without secret leakage.
- Scheduled backups resume successfully after restore.
