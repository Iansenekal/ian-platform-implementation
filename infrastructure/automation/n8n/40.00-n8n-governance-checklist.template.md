# 40.00 n8n Governance Checklist

- Workflow catalog maintained with owner, purpose, and data classification.
- Service credentials are scoped per integration and environment.
- No secrets in workflow JSON exports or repository files.
- Outbound internet calls are disabled unless explicitly approved.
- Admin access enforces SSO + MFA.
- Deny tests confirm unscoped access is blocked.
- Backup and restore tests include representative workflows.
