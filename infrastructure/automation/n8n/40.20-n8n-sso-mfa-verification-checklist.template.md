# 40.20 n8n SSO/MFA Verification Checklist

- OIDC SSO login works and IdP groups map to n8n roles.
- MFA is enforced for n8n admin role and configured operator/builder roles.
- RBAC deny tests block non-admin users from admin settings.
- Session and idle timeout controls enforce re-authentication for sensitive actions.
- Break-glass process exists with dual-control storage and post-use rotation.
- Audit events include auth, role changes, workflow changes, and credential changes.
