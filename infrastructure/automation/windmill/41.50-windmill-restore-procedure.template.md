# 41.50 Windmill Restore Procedure

1. Confirm incident scope and target environment.
2. Validate secret and encryption dependency recovery readiness.
3. Restore Windmill database from last known-good backup.
4. Restore Windmill config and reverse proxy settings as needed.
5. Start Windmill services and verify health + LAN UI reachability.
6. Revalidate SSO/MFA and RBAC role mapping.
7. Execute representative jobs and verify safe logging controls.
8. Execute post-restore verification checklist and capture evidence.
