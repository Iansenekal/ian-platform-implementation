# 41.00 Windmill Governance Checklist

- [ ] Job catalog maintained (owner, purpose, data classification)
- [ ] Script review required for privileged jobs
- [ ] No secrets in scripts or export artifacts
- [ ] SSO + MFA enforced for admins/operators per policy
- [ ] RBAC separation active (Admin, Operator, Builder, Viewer)
- [ ] Outbound internet calls disabled by default
- [ ] Backup and restore test evidence exists
