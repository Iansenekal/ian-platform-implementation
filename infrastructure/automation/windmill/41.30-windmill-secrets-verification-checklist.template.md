# 41.30 Windmill Secrets Policy Verification Checklist

- [ ] No secrets appear in scripts, repos, docs, or job outputs
- [ ] Host secret files exist with required permissions (`600 root:root`)
- [ ] Windmill secret references are scoped and not exposed to unauthorized roles
- [ ] Secret inventory is current and matches active integrations
- [ ] Least-privilege tests pass for all service accounts
- [ ] Rotation schedule is evidenced and current
- [ ] Secret exposure incident tabletop has been completed
