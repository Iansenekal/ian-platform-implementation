# 41.50 Windmill Post-Restore Verification Checklist

- [ ] Windmill service remains healthy after restart
- [ ] LAN-only access and allowlist controls remain enforced
- [ ] SSO login and admin MFA enforcement work
- [ ] RBAC deny tests pass for privileged settings/jobs
- [ ] Script/flow catalog and schedules are present
- [ ] Representative least-privilege job executes successfully
- [ ] Logs/audit do not expose secrets or sensitive payloads
- [ ] Scheduled backups resume after restore
