# 82.50 Evidence Checklist

Store this evidence for each model change and each weekly audit:

- Approved change record (`CHG-YYYY-####`) under `/opt/ollama/config/approvals/`
- Allowlist diff for `/opt/ollama/config/model-allowlist.yaml`
- Wrapper execution evidence (`82.50-allowlist-enforced-pull.sh` output)
- Inventory record JSON under `/opt/ollama/config/model-inventory/`
- Installed model export (`ollama list`)
- Weekly audit report from `82.50-allowlist-audit.sh`
- Proof outbound internet remained blocked except approved maintenance window

Retention target: 180 days minimum.
