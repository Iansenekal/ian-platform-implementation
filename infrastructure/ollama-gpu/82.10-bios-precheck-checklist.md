# 82.10 BIOS/UEFI + GPU Precheck Checklist

- [ ] BIOS/UEFI updated to approved stable version.
- [ ] VT-x and VT-d/IOMMU enabled.
- [ ] Above 4G Decoding enabled.
- [ ] ReBAR enabled (or documented exception).
- [ ] GPU installed in primary x16 slot.
- [ ] PCIe policy set (`auto` or `gen4` fallback documented).
- [ ] UEFI boot mode confirmed.
- [ ] Secure Boot policy chosen and documented (`A` or `B`).
- [ ] PSU/GPU/RAM/CPU inventory captured.
- [ ] BIOS setting screenshots captured and archived.
- [ ] Post-install stability plan acknowledged (30-60 min sustained load check).
