# 71.40 Mail Audit Verification Checklist

- Verify event chain includes NOTIF_CREATED, EMAIL_SENT, ACTION_TOKEN_ISSUED.
- Verify approve flow emits ACTION_LINK_CLICKED, ACTION_AUTH_SUCCESS, APPROVAL_DECISION_APPROVED.
- Verify replayed token emits TOKEN_REPLAY_BLOCKED.
- Verify expired token emits TOKEN_EXPIRED.
- Verify logs exclude token values, SMTP credentials, message body, and attachments.
- Verify correlation_id is present across workflow, mail, and decision events.
