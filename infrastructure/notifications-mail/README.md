# Notifications and Mail Infrastructure Artifacts

This directory contains baseline templates and verification assets for `71.*` notifications and mail architecture rollout items.

Current baselines:
- `71.10` LAN-only mail architecture and relay/security model.
- `71.15` actionable approval email controls (Approve/Reject links, token/MFA/security requirements).
- `71.40` notifications/mail audit-event taxonomy, schema, and retention controls.
- `71.50` mail evidence retention and purge/legal-hold controls.
