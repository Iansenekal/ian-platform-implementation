# 30.70 Retention and Privacy Regression Checklist

- [ ] Metadata-only default enforced for newly onboarded sources
- [ ] Sensitive exclusions do not appear in search results or mind-map
- [ ] Retention class assigned to indexed records and KG objects
- [ ] Purge run executed and summary captured (counts + scope)
- [ ] Deletion propagation SLA test completed with known sample files
- [ ] ACL narrowing reflects in visibility without stale overexposure
- [ ] Person/entity enrichment remains disabled unless approved
- [ ] Query logging level matches approved privacy profile
- [ ] Evidence stored in approved repository
