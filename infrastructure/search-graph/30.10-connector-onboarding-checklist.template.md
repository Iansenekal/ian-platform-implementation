# 30.10 Connector Onboarding Checklist

- [ ] Source name, owner, and business justification documented
- [ ] Include/exclude scope approved (paths/sites/libraries)
- [ ] Minimization mode selected (metadata-only or approved content extraction)
- [ ] Sensitive exclusions confirmed (Finance/Legal/HR as applicable)
- [ ] ACL proof plan defined with deny-test sampling
- [ ] Dedicated non-human service identity created
- [ ] Secrets stored in `/opt/<stack>/secrets` only
- [ ] LAB first crawl executed with low concurrency
- [ ] ACL deny tests passed (no unauthorized leakage)
- [ ] Run summary captured (scanned/indexed/skipped/failed)
- [ ] Incremental schedule + retry/backoff configured
- [ ] change control approved for PROD promotion
