# 30.90 Voice Transcript Indexing and Graph Linking Verification Checklist

- Restricted transcript in Nextcloud is searchable only by authorized principals.
- Unauthorized user graph view shows no transcript node and no inferred hidden edges.
- Authorized user sees transcript node with allowed related project/document/topic links.
- Relationship reason is explainable (explicit reference or semantic similarity).
- Retention purge removes transcript from index and graph references.
- Audit events exist for indexing pipeline and transcript viewer access (if enabled).
